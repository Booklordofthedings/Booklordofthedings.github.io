Everything about my website

I finallly “finished” my website

I dont think I will ever truly finish it, as even with this setup right now I am immideatly thinking about making a custom static page generator

The Idea

I always really liked the idea of having a website/blog where I can post my own stuff, while not being limited by lenght like on twitter, but also by being more open than a private discord server. And since Computer science in general is one of my favourite thing to learn about/do stuff with I decided to try to do this.

Prior to this project my main knowledge was that the web uses html css and js for displaying websites and had actuallly used each of these Things before. Also there was this program called Wordpress that could be used to host websites that only needed some textfile input.

After some conversation in a discord server I got pointed to a css framework called Bootstrap wich was able to make pages look good on both mobile and desktop due a responsive design principle.

my first try at a website

I never ended up finishing this one, but I learned alot about responsive web development and some more general rules. In addition to that I started learning a bit about how modern websites are created with content management systems. But there always was this idea in my head of making a static page and writing an html generator, even before I knew what theses where.

my Idea for something that I could write an autogenerator for

After asking on twitter the user Cyberpanda told me about a static site generator named hugo wich actually was pretty easy and confortable to use. So the idea of making my own sst isnt fully of the table, but for the first steps just using a modified theme is fine.

This page uses the Piercer Theme with some modification to look more how I want it too look on desktop devices.

The hosting for this site was a problem aswell. Netcup, my initial idea didnt actually have a whois protection for their pages and while netcup was fine there was an issue with the cost going insanely high after the first year.

After asking the user Cadey A. Ratio they suggested for me to just use a domain and host this page on github pages, wich is actually what I ended up doing.

In the end I went with Booklordofthe.dev as there wasnt any real way to perfectly finish my name, but with this it fit my style while not being too far off from what I would use.
