Floppy Control

February 6, 2021

The only video game project I ever finished

I actually finished a game at some point

My original goal since elementary school always was, to become a game programmer, and while currently the focus changed quite a bit to general programming with gamedev as a sidehobby I did learn alot of game programming and in the summer of 2020 I actually finished my first game project for a GameJam.
(A gamejam is a time limited game making “competition” where the participants make a game based on a certain theme)

It is downloadable here

The gamejam was the Game makers toolkit gamejam with the theme being: “out of control”

My first idea was a stealth game where the user would have to direct a thief apprentice and help him, as the apprentice himself was incompetent (You will probably be able to find some tweets about this on my twitter acc). This idea is still one I like, however programming a ai in this little time (48h) as a solo dev was not feasable.

The engine I was using was unity, and unity is pretty good at making 2d platformers, wich is actually why I decided to do something like this. “Out of control” is an interesting theme so I decided to make the player character not directly controlable, but instead only through floppy disks you can place that will trigger an action if the bot touches them.

So the entire game ended more up like a puzzler, so the immideate puzzle ideas came to mind:

  • Spikes
  • Crumbling Platforms
  • Deathpits
  • Jumppads

I just realized how little these actually are, so im actually kinda impressed about myself creating a pretty good game on such a little basis.

The actions the bot could take where:

  • Jump (Propell upwards)
  • Turn around
  • Dash into the direction you are looking
  • Cancel all momentum

The main thing I learned here is that for design the thing that matters most is how mechanics are used and not the amount of mechanics.

Also I learned that learning a new system for a time-limited thing is bad (I had alot of time wasted by trying to make the unity input system work)

I ended up getting ~580/5000 in terms of fun wich is honestly pretty awesome.

At some point in the future I might actually make a full version of the game
